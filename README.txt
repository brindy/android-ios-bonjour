
The Android app will start and broadcast a service.  It must run on a device, not an emulator, for the iOS app to be able to connect to it.

The iOS app will search for services that match the service type.  You can pull to refresh to force a new search.

Tap on a discovered service to initiate communication.  Check the logs for the confirmation.

This is not robust - it's just a quick spike to show that it's possible.
